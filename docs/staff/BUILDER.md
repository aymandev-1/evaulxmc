# Builder Guide

You build and shape the server's worlds. You are **not** a moderator — keep to build
tools and leave punishments to the mod team. Read [STAFF_INTRO](STAFF_INTRO.md) first.

## Your panel: `/builderpanel`
Everything below in one menu — gamemode, fly, build mode, time/weather, movement,
and the builder announcement.

## Build tools
| Command | Use |
|---|---|
| `/gamemode creative` (`/gmc`) | Switch to creative to build. |
| `/gamemode survival` | Back to survival to test. |
| `/fly` | Toggle flight. |
| `/buildmode` | Bypass lobby protection so you can edit protected areas. |
| `/speed <1-10>` | Move faster while building. |
| `/hat` | Wear your held block as a hat (display/preview). |
| `/craft` | Open a workbench anywhere. |
| `/top` / `/up` | Jump to the surface / straight up. |

## World tools
| Command | Use |
|---|---|
| `/time day` / `/time night` | Light your build for screenshots. |
| `/weather clear` | Stop rain ruining a shot. |
| `/nightvision` | See in dark interiors. |

## Sharing your work
- `/builderannounce <message>` — tell the server about a new build or event area.

## Watch out for
- Build mode lets you edit protected lobbies — turn it **off** when you're done.
- Don't hand out items to players from creative; that's not your role.
- Coordinate big terrain changes with an Admin first.
