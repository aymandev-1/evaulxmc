# Server Changelog (Staff Copy)

> ⚠️ **The server is in testing.** Features may change, reset, or go offline without notice
> while we polish things. Don't treat anything here as final, and warn players that data may be wiped.

A short, curated list of the features that actually change what players experience.
Keep this handy so you can answer "what's new?" in chat. Internal/technical changes are left out on purpose.

---

## Latest

**Content Creator program is live**
- Creators can now go live (`/golive`), and the whole server gets a title + announcement.
- Shoutouts (`/shoutout`), milestones, and social broadcasts so players can find creators.
- Creator **giveaways** (`/ccgiveaway`) hand real ranks to random online players.
- Subscriber **redeem codes** (`/redeemcode <code>`) give players free rewards.
- These now work the moment a creator is granted — no setup needed.

**Cosmetic tags refreshed**
- The `/tag` menu is cleaner and every tag now shows as a colour-matched icon.
- Players pick, preview, and equip tags that show in chat.

**Role control panels**
- One-click menus for each role: `/modpanel`, `/builderpanel`, `/developerpanel`, `/creatorpanel` (plus existing `/adminpanel`, `/ownerpanel`, `/staffpanel`).
- Faster moderation = fewer rule-breakers slipping through.

---

## Core player features (already on the server)

- **Ranks & tags** — visible prefixes, colours, and cosmetic chat tags.
- **Economy** — coins, `/pay`, `/daily` reward.
- **Teleporting** — `/spawn`, `/home` + `/sethome`, `/warp`, `/tpa`.
- **Social** — `/friend`, `/party`, `/msg` + `/reply`, `/mail`.
- **Reports & help** — players can `/report` rule-breakers and `/helpop` for staff help.
- **Appeals** — banned/muted players can `/appeal`.
- **Quality of life** — `/afk`, `/near`, `/coords`, `/ping`, `/playtime`.

> Tip: announce only the headline items in-game. Players care about what they can *do*,
> not the command list — point them at `/help` for the rest.
