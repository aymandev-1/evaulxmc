# Content Creator Guide

You stream/record on the server and help bring players in. These tools are unlocked the
moment an admin runs `/creator grant <you>` — no extra permission setup needed.

## Your panel: `/creatorpanel`
Every tool below in one menu.

## First: set up your profile
Players only see your links if you set them. Use `/creator set <field> <value>`:
- `youtube`, `twitch`, `tiktok`, `twitter`, `instagram`, `discord`
- `displayname`, `description`, `streamtitle`
- `code` — your subscriber **redeem code** (players run `/redeemcode <code>` for rewards)

Check it with `/creator profile`.

## Going live
| Command | What players see |
|---|---|
| `/golive [platform]` | Server-wide title + announcement that you're streaming, with your link. |
| `/offair` | Ends your live status. |
| `/socials` | Broadcasts your links and stream info. |

## Engaging players
| Command | Use |
|---|---|
| `/shoutout <player>` | Hype up a viewer in front of everyone. |
| `/ccgiveaway` | Give a real rank to a random online player. |
| `/milestone <message>` | Announce a sub/follower milestone. |
| `/spotlight <player>` | Shine a server-wide spotlight on someone. |
| `/recording` | Mark yourself as recording (so staff/players know). |
| `/watchparty <start|end>` | Run a watch-party event. |
| `/cchat <message>` | Private chat with other creators. |
| `/ccannounce <message>` | Creator-styled announcement. |

## Etiquette
- Giveaways and shoutouts have cooldowns — don't spam them.
- Keep stream titles/descriptions server-appropriate.
- Use streamer mode (`/streamermode`) if you want to hide your real IGN on stream.
